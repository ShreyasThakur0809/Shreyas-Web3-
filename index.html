Execution & Transactions

How instructions are bundled and executed differs significantly, impacting developers and users.

Solana

Transaction Model

A transaction is a bundle of one or more atomic instructions. All instructions succeed or fail together.

Instruction 1: Create a data account (PDA) for a user.
Instruction 2: Initialize data in that account.
Instruction 3: Transfer SOL to the new account for rent.

This atomicity allows for complex, multi-step operations in a single, secure transaction.

Ethereum

Transaction Object

A transaction is a single message from an EOA to another account, containing key fields.

  • To: Contract or EOA address.
  • Value: Amount of ETH to send.
  • Data: Encoded function call and arguments.
  • Gas Limit/Price: Fee parameters.
  • Nonce: Transaction counter.

The EVM executes the code based on the `data` field, modifying the contract's internal storage.

dApp Development Concepts

Key tools and concepts for building frontends that interact with blockchains.

React Hooks: `useState` & `useEffect`

These are the building blocks of modern React applications for managing state and side effects.

`useState`

Manages component data. When state changes, the component re-renders.

`useEffect`

Handles side effects like data fetching or subscriptions after the component renders.

Example: Fetching Data

const [data, setData] = useState([]); useEffect(() => { // Runs once after initial render fetch('api/data') .then(res => res.json()) .then(json => setData(json)); }, []); // Empty array means run once

Ethereum: ABI

The Application Binary Interface (ABI) is the "API documentation" for an Ethereum smart contract.

It's a JSON file that defines how to call contract functions and interpret their data, enabling tools and UIs to interact with them.

Example ABI Snippet for `add(uint a, uint b)`

{ "inputs": [ { "name": "a", "type": "uint256" }, { "name": "b", "type": "uint256" } ], "name": "add", "outputs": [ { "name": "", "type": "uint256" } ], "stateMutability": "pure", "type": "function" }
