Execution & Transactions

How instructions are bundled and executed differs significantly, impacting developers and users.

Solana

Transaction Model

A transaction is a bundle of one or more atomic instructions. All instructions succeed or fail together.

Instruction 1: Create a data account (PDA) for a user.
Instruction 2: Initialize data in that account.
Instruction 3: Transfer SOL to the new account for rent.

This atomicity allows for complex, multi-step operations in a single, secure transaction.

Ethereum

Transaction Object

A transaction is a single message from an EOA to another account, containing key fields.

  • To: Contract or EOA address.
  • Value: Amount of ETH to send.
  • Data: Encoded function call and arguments.
  • Gas Limit/Price: Fee parameters.
  • Nonce: Transaction counter.

The EVM executes the code based on the `data` field, modifying the contract's internal storage.

dApp Development Concepts

Key tools and concepts for building frontends that interact with blockchains.

React Hooks: `useState` & `useEffect`

These are the building blocks of modern React applications for managing state and side effects.

`useState`

Manages component data. When state changes, the component re-renders.

`useEffect`

Handles side effects like data fetching or subscriptions after the component renders.

Example: Fetching Data

const [data, setData] = useState([]); useEffect(() => { // Runs once after initial render fetch('api/data') .then(res => res.json()) .then(json => setData(json)); }, []); // Empty array means run once

Ethereum: ABI

The Application Binary Interface (ABI) is the "API documentation" for an Ethereum smart contract.

It's a JSON file that defines how to call contract functions and interpret their data, enabling tools and UIs to interact with them.

Example ABI Snippet for `add(uint a, uint b)`

{ "inputs": [ { "name": "a", "type": "uint256" }, { "name": "b", "type": "uint256" } ], "name": "add", "outputs": [ { "name": "", "type": "uint256" } ], "stateMutability": "pure", "type": "function" }

Send SOL (Solana)

import { useConnection, useWallet } from "@solana/wallet-adapter-react"; import { LAMPORTS_PER_SOL, PublicKey, SystemProgram, Transaction } from "@solana/web3.js"; function SendTokens() { const wallet = useWallet(); const {connection} = useConnection(); async function sendTokens() { const to = "RecipientPublicKeyHere"; // Replace with actual recipient const amount = 1; // Amount in SOL if (!wallet.publicKey) return; const transaction = new Transaction().add( SystemProgram.transfer({ fromPubkey: wallet.publicKey, toPubkey: new PublicKey(to), lamports: amount * LAMPORTS_PER_SOL, }) ); await wallet.sendTransaction(transaction, connection); console.log("Sent " + amount + " SOL to " + to); } return ; }

Request Airdrop (Solana)

import { useConnection, useWallet } from "@solana/wallet-adapter-react"; import { LAMPORTS_PER_SOL } from "@solana/web3.js"; function RequestAirdrop() { const wallet = useWallet(); const { connection } = useConnection(); async function requestAirdrop() { const amount = 2; // Amount in SOL if (!wallet.publicKey) return; await connection.requestAirdrop(wallet.publicKey, amount * LAMPORTS_PER_SOL); console.log("Airdropped " + amount + " SOL"); } return ; }

Sign Message (Solana)

import { useWallet } from '@solana/wallet-adapter-react'; import { ed25519 } from '@noble/curves/ed25519'; import bs58 from 'bs58'; function SignMessage() { const { publicKey, signMessage } = useWallet(); async function onClick() { if (!publicKey || !signMessage) return; const message = "Hello Solana!"; const encodedMessage = new TextEncoder().encode(message); const signature = await signMessage(encodedMessage); if (!ed25519.verify(signature, encodedMessage, publicKey.toBytes())) { throw new Error('Message signature invalid!'); } console.log(`Signature: ${bs58.encode(signature)}`); } return ; }

Send ETH (Ethereum)

// Requires a connected wallet and ethers.js loaded async function sendEth() { if (typeof window.ethereum === 'undefined' || !window.ethereum.selectedAddress) { console.log("Ethereum wallet not connected."); return; } const provider = new ethers.providers.Web3Provider(window.ethereum); const signer = provider.getSigner(); const toAddress = "0xRecipientAddressHere"; // Replace with actual recipient const amountEth = "0.001"; // Amount in ETH try { const tx = { to: toAddress, value: ethers.utils.parseEther(amountEth) // Converts ETH to Wei }; const transactionResponse = await signer.sendTransaction(tx); console.log("Transaction sent:", transactionResponse.hash); await transactionResponse.wait(); // Wait for transaction to be mined console.log("Transaction confirmed!"); } catch (error) { console.error("Failed to send ETH:", error); } } // To use:
